При решении проблем рассматривают два аспекта выработки решения:
- поиск решения - процедура поиска путей достижения цели, разработкой альтернатив
- принятие/выбор решения - процедуры выбора ЛПР конкретной альтернативы из всех альтернатив с учетом целей, ограничений.
Принятие решения происходит во времени, поэтому вводится понятие процесса принятия решения.
Три этапа процесса ПР:
- постановка задачи
- поиск решения
- выбор решения

Постановка задачи предполагает:
- выявление и описание проблемной ситуации
- определение необходимых для принятия решений ресурсов
На этапе поиска решения:
- анализ проблемной ситуации
- формирование целей
- определение ограничений
- генерация альтернатив
- определение предпочтений
На этапе выбора решений:
- определение множества допустимых решений
- формирование критериев выбора решений
- определение эффективных решений
- выбор наилучшего решения

При принятии решений характерен недостаток информации для осуществления рационального выбора, имеет место неопределенность.
Источники неопределенности может иметь различную природу - например, неоднозначность начального описания проблемной ситуации
Неопределенность обусловлена следующими причинами:
- неточность измерений параметров
- недостаток информации о исходной ситуации
- поведение среды
- неизвестные последствия выбора
Другой источник неопределенности - естественный язык - лингвистическая неопределенность. Для учета этой неоднозначности используются модели принятия решений, основанных на теории нечетких множеств.
