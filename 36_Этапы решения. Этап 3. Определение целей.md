На основе сформулированной проблемы определяется цель - направление в котором надо двигаться, чтобы решить проблему
На данном этапе конкретизируется цель, формулируется в виде конкретных условий.
Для правильного выделения целей необходимо уточнить миссию организации, ее генеральную цель, построить дерево целей.
Дерево строится путем расчленения главной цели на подцели. Затем каждая из подцелей рассматривается как цель и расчленяется на подцели более низкого уровня.  Эта процедура осуществляется до тех пор, пока не будет получен полный список локальных целей, для которых известны средства их достижения.
Если все подцели и связи между ними представить графически, то получим картину, напоминающую дерево. На самом верхнем уровне расположена главная цель, а на самых нижних уровнях - средства достижения локальных целей. Это метод сверху вниз.
Альтернативный метод - снизу вверх - имеется множество локальных целей, и на основе их группировок, учета взаимосвязей определяется главная цель, которую можно достичь, реализуя локальные цели.

Несмотря на то, что дерево целей формируется на эвристической основе, при реализации этого метода можно выделить две стадии:
- построение первоначального варианта дерева целей
- определение коэфов относительной важности его отдельных элементов и формирование окончательного дерева.

Основной подход к построению дерева целей - метод структуризации, он основан на дезагрегировании (поэтапном расчленении) исследуемой проблемы на элементы с последующей численной оценкой их важности. Поэтапное разделение проводится до тех пор, пока не будет достигнут требуемый уровень детализации.

Метод структуризации позволяет установить полный набор элементов дерева целей на каждом уровне и определить взаимосвязи между ними. 
Принципы детализации дерева:
- Объектовый принцип - элементы дерева разбиваются на элементы той же природы, более детализированные
- Функциональный принцип - в дереве определяется содержание тех функций, которые должны выполняться для достижения цели
- Принцип детализации
- Принцип охвата всех факторов влияющих на решение рассматриваемой проблемы
- Принцип адресности
- Системный принцип

После того, как цели сформулированы, появляется возможность выбора связанных с ними критериев.
